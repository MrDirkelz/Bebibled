# Bebibled Goal

This project is a bejeweled, bookworm adventures inspired game. The intention is to increase biblical games, edification on the Google Play Store.

## Moral Choice

This project is not intended to create revenue, and is solely a passion project. God will provide in other ways, right now, this is what he has worked in my heart to do.

### Tech stack:
- Gamemaker studio 2
- Claude + Codex for development effeciency

### Technical details:
- I have decided to use the KJV bible for now, as it is in the public domain, and available freely.
- Utilize the 3D Array Structure for the json file for the KJV bible: The data is structured as a clear [book][chapter][verse] multi-dimensional array. This means this game can easily pull specific strings via localized index variables (e.g., bible[0][0][0] for Genesis 1:1).
